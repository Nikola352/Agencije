# WD-SV-9-2022

Projektni zadatak iz predmeta <i>Veb dizajn</i> za 2023. godinu, SIIT.

Front-end dio za veb-sajt koji objedinjuje turističke agencije.


## Korištene tehnologije

<ul>
    <li>React - front-end framework</li>
    <li>Vite - build tool</li>
    <li>TailwindCSS - utility-first CSS framework</li>
    <li>Framer Motion - animation library for react</li>
    <li>Fuse.js - fusy search library</li>
    <li>Floating UI - UI library for positioning floating elements</li>
</ul>

## Pokretanje 

Klonirajte repozitorijum i pokrenite komande:

```
npm install
npm run dev
```
